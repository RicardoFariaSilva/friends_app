# FriendsApp
Basic Elixir project made in my learning process.

A CRUD CLI to add friends on your notepad. Basically, it manipulates a CSV file in order to record registries.

## How it works?

### First you are going to receive a welcome message:

<img width="475" alt="image" src="https://user-images.githubusercontent.com/17459963/126174010-29035553-74ea-4d33-aa6f-2d3d82b2c8da.png">

### Then the notepad menu appears:
Here you can choose the basic CRUD options.

<img width="482" alt="image" src="https://user-images.githubusercontent.com/17459963/126174040-59f871b5-d3a8-4b78-82f5-aac3fcde3090.png">

### Create
Typing `1` you receive a confirmation message.

`Y` -> Continue to add page  
`n` -> Back to menu

<img width="472" alt="image" src="https://user-images.githubusercontent.com/17459963/126174719-a38d1df2-270b-4f70-8630-d569b315d135.png">

In the add page you can digit your friend's name, email and phone.

<img width="473" alt="image" src="https://user-images.githubusercontent.com/17459963/126174799-b37dacd1-9f4e-4785-b90a-d128f23d17be.png">

### Read
Back to Menu, typing `2` you receive another confirmation message to list your friends.

`Y` -> Continue to list page  
`n` -> Back to menu

<img width="473" alt="image" src="https://user-images.githubusercontent.com/17459963/126174912-9bebbd74-8a3c-41cd-8b25-6b1aee88fe9d.png">

This page lists all your saved friends.

<img width="748" alt="image" src="https://user-images.githubusercontent.com/17459963/126174975-e2cc474c-c0c5-417d-a0a0-6378dec7a782.png">

### Update
Back to Menu, typing `3` there is a confirmation message.

`Y` -> Continue to find friend page  
`n` -> Back to menu

<img width="472" alt="image" src="https://user-images.githubusercontent.com/17459963/126175112-3ef10003-4afd-49bd-9df0-948c92f13e98.png">

Email is needed to find friend's data.

<img width="476" alt="image" src="https://user-images.githubusercontent.com/17459963/126175189-cc7accde-4f60-4417-b036-9cea9b1affcc.png">

If no friend is found, this message is shown.

<img width="472" alt="image" src="https://user-images.githubusercontent.com/17459963/126175443-746c21e1-0be7-49dc-bad1-8a14988accca.png">

But if email is correct, you may edit the data.

<img width="472" alt="image" src="https://user-images.githubusercontent.com/17459963/126175242-2941339a-8fe2-4876-bd4a-a9c9431ce6a2.png">

A confirmation message with your friend's data appears.

`Y` -> Continue to update page  
`n` -> Back to menu

<img width="473" alt="image" src="https://user-images.githubusercontent.com/17459963/126175308-9f80f090-6943-4caf-818f-b412d2c2fbd7.png">

At this page you can type the new data.

### Delete
Back to Menu, typing `4` there is a confirmation message again.

<img width="472" alt="image" src="https://user-images.githubusercontent.com/17459963/126175351-2cf2d995-6158-4c5e-a7da-a91d2a19b76a.png">

`Y` -> Continue to find friend page  
`n` -> Back to menu

<img width="475" alt="image" src="https://user-images.githubusercontent.com/17459963/126175385-62f35584-d820-428a-a0c5-9fa9517da2ea.png">

Email is needed to find your friend.

<img width="472" alt="image" src="https://user-images.githubusercontent.com/17459963/126175443-746c21e1-0be7-49dc-bad1-8a14988accca.png">

Again, if there is no friend with this email, you receive an error message.

<img width="473" alt="image" src="https://user-images.githubusercontent.com/17459963/126175526-f6974dae-75e4-45f6-95df-bf10133a871d.png">

But if it's correct, you can delete it.
